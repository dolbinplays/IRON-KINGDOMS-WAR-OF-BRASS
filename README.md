# Iron Kingdoms: War of Brass — Premium Battle Art + Audio Branch v0.26.06.10.1912

This is a GitHub Pages-ready folder build. Upload the contents of this folder to a GitHub repository and enable Pages from the repository settings.

## Play

Open `index.html` in a browser, or host the folder with GitHub Pages. The game still uses Discord JSON handoff: finish a turn, save the JSON, and post it to Discord for the next player.

## What's included

- `index.html`
- `/assets/units/` faction-colored PNG unit sprites
- `/assets/commanders/` commander sprites, battle sprites, and portraits
- `/assets/tiles/` 64x64 PNG terrain tiles
- `/assets/structures/` faction-colored structures
- `/assets/effects/` combat effect PNGs
- `/assets/battle_backgrounds/` zoomed battle terrain backgrounds
- `/assets/audio/` generated original retro WAV loops and fanfares
